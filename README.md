## 📖 Overview

**MisXMatch** is an AI-powered Missing Person Identification and Tracking System designed to help reunite missing individuals with their families through a centralized digital platform.

The platform enables **Citizens, Police Departments, Hospitals, NGOs, and Administrators** to collaborate efficiently using AI-assisted face matching, CCTV analysis, case management, and real-time updates.

---

## 🎯 Problem Statement

Missing person investigations are often delayed because information is scattered across different organizations.

Current systems rely heavily on manual searching, making it difficult to identify missing individuals quickly.

**MisXMatch** solves this problem by providing a centralized platform where multiple organizations can securely collaborate while AI assists in identifying potential matches.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/MisXMatch.git
```

Go to project folder

```bash
cd MisXMatch
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```
